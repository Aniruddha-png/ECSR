ECSR (Emotion Classification and Sentiment Recognition) is a multimodal emotion detection system that combines information from multiple sources, including facial expressions, speech/audio features, and textual content, to identify human emotions more accurately than single-modal approaches. The project leverages machine learning and deep learning techniques to fuse different modalities and improve emotion recognition performance in real-world scenarios.

Key Features:
• Facial emotion recognition from images/video streams
• Speech emotion recognition from audio signals
• Text sentiment and emotion analysis
• Multimodal feature fusion for enhanced accuracy
• Real-time emotion prediction
• Comprehensive performance evaluation using accuracy, precision, recall, F1-score, and confusion matrices

Technologies:
Python, OpenCV, TensorFlow/Keras, PyTorch, NLP libraries, Audio Processing, Machine Learning, Deep Learning
